What do users see when there's a Javascript error on webpage?
"Nothing!"

As a programmer, we need to follow certain rules which are listed below,
  • Assume your code will fail
  • Log errors to the server
  • You, not the browser, handle errors
    • try-catch
    • window.onerror
  • Identify where errors might occur
    • Type coercion errors
    • Data type errors
    • Communication errors



  • Throw your own errors
    • Errors should be thrown in the low-level parts of the application
    • Use try-catch blocks in higher level parts.

  • Distinguish fatal vs non-fatal




  • Provide a debug mode
    • Assign a variable that is globally available
    • try-catch should re-throw the error
    • window.onerror should return false
    • Allow the browser to handle the error
